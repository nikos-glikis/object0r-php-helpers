# object0r-php-helpers
Php Helpers Collection

To install add in composer installer:

      "repositories": [
      {
        "type": "vcs",
        "url": "git@github.com:nikos-glikis/object0r-php-helpers.git"
      }
    ],
    
    "require": {
      ...
      "nikosglikis/object0r-php-helpers": "^1.0"
    }
